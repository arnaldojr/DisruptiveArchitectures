def aplicar_convolucao_manual(imagem, kernel):
    """Aplica convolu√ß√£o manualmente para demonstra√ß√£o"""
    img_h, img_w = imagem.shape
    kernel_h, kernel_w = kernel.shape
    
    # Tamanho da sa√≠da
    out_h = img_h - kernel_h + 1
    out_w = img_w - kernel_w + 1
    
    resultado = np.zeros((out_h, out_w))
    
    for i in range(out_h):
        for j in range(out_w):
            # Regi√£o da imagem
            regiao = imagem[i:i+kernel_h, j:j+kernel_w]
            # Produto elemento a elemento + soma
            resultado[i, j] = np.sum(regiao * kernel)
    
    return resultado

    üéØ O que √© Pooling?¬∂

    O pooling √© uma opera√ß√£o de subsampling que:

    • üìâ Reduz dimensionalidade dos feature maps
    • üéØ Mant√©m caracter√≠sticas importantes
    • ‚ö° Diminui custo computacional
    • üõ°Ô∏è Adiciona invari√¢ncia a pequenas transla√ß√µes
    • üö´ Reduz overfitting

    üîç Tipos de Pooling¬∂

    1Ô∏è‚É£ Max Pooling (Mais Comum)¬∂

    • Opera√ß√£o: Seleciona o valor m√°ximo na janela
    • Intui√ß√£o: Preserva as caracter√≠sticas mais ativas
    • Uso: Detec√ß√£o de bordas, texturas

    2Ô∏è‚É£ Average Pooling¬∂

    • Opera√ß√£o: Calcula a m√©dia dos valores na janela
    • Intui√ß√£o: Suaviza e reduz ru√≠do
    • Uso: Menos comum, algumas arquiteturas espec√≠ficas

    3Ô∏è‚É£ Global Average Pooling¬∂

    • Opera√ß√£o: M√©dia de todo o feature map ‚Üí 1 valor
    • Vantagem: Substitui camadas Dense finais
    • Benef√≠cio: Reduz drasticamente overfitting

def max_pooling_manual(imagem, pool_size=2, stride=2):
    """Implementa√ß√£o manual do max pooling para demonstra√ß√£o"""
    h, w = imagem.shape
    out_h = (h - pool_size) // stride + 1
    out_w = (w - pool_size) // stride + 1
    
    resultado = np.zeros((out_h, out_w))
    
    for i in range(out_h):
        for j in range(out_w):
            h_start = i * stride
            h_end = h_start + pool_size
            w_start = j * stride  
            w_end = w_start + pool_size
            
            # Max pooling
            resultado[i, j] = np.max(imagem[h_start:h_end, w_start:w_end])
    
    return resultado

def avg_pooling_manual(imagem, pool_size=2, stride=2):
    """Implementa√ß√£o manual do average pooling"""
    h, w = imagem.shape
    out_h = (h - pool_size) // stride + 1
    out_w = (w - pool_size) // stride + 1
    
    resultado = np.zeros((out_h, out_w))
    
    for i in range(out_h):
        for j in range(out_w):
            h_start = i * stride
            h_end = h_start + pool_size
            w_start = j * stride
            w_end = w_start + pool_size
            
            # Average pooling
            resultado[i, j] = np.mean(imagem[h_start:h_end, w_start:w_end])
    
    return resultado

    üèóÔ∏è Estrutura Geral¬∂

    Uma CNN completa √© composta por duas partes principais:

    1Ô∏è‚É£ Extrator de Caracter√≠sticas (Feature Extractor)¬∂

    • Fun√ß√£o: Detectar padr√µes visuais hier√°rquicos
    • Componentes: Conv2D + Pooling + Ativa√ß√£o
    • Processo: Bordas ‚Üí Texturas ‚Üí Formas ‚Üí Objetos
    • Sa√≠da: Feature maps com caracter√≠sticas extra√≠das

    2Ô∏è‚É£ Classificador (Classifier)¬∂

    • Fun√ß√£o: Tomar decis√£o baseada nas caracter√≠sticas
    • Componentes: Flatten + Dense layers (MLP)
    • Processo: Features ‚Üí Combina√ß√µes ‚Üí Probabilidades
    • Sa√≠da: Classifica√ß√£o final

    üéØ Arquitetura Visual¬∂

    üîÑ Fluxo de Processamento¬∂

    Imagem ‚Üí [Conv‚ÜíReLU‚ÜíPool]√óN ‚Üí Flatten ‚Üí [Dense‚ÜíReLU]√óM ‚Üí Softmax ‚Üí Classes
  ‚Üì             ‚Üì                ‚Üì            ‚Üì              ‚Üì
 Raw         Features         Vector      Hidden        Probabilities
Pixels      Extraction      Formato 1D   Layers        por Classe

    üßÆ Hierarquia de Caracter√≠sticas¬∂

    | Camada | Detecta | Exemplo | |--------|---------|---------| | Conv1 | üîç Bordas, linhas | /, \, ‚Äî, | | | Conv2 | üî∫ Formas simples | Cantos, curvas | | Conv3 | üéØ Partes de objetos | Olhos, rodas, janelas | | Conv4+ | üñºÔ∏è Objetos completos | Faces, carros, casas |

    ‚ö° Por que essa Divis√£o?¬∂

    Extrator (Convolucional):

    • ‚úÖ Preserva informa√ß√£o espacial
    • ‚úÖ Detecta padr√µes locais
    • ‚úÖ Invariante √† posi√ß√£o
    • ‚úÖ Compartilha pesos

    Classificador (Dense):

    • ‚úÖ Combina informa√ß√£o global
    • ‚úÖ Aprende rela√ß√µes complexas
    • ‚úÖ Produz probabilidades
    • ‚úÖ Decis√£o final
